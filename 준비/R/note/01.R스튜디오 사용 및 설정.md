# R스튜디오 사용 및 설정 

## 폰트 설정
1. Tools
2. Global Options
3. Appearance
4. Editor font size : 20
5. Zoom : 150%  (Ctrl + [+], Ctrl + [-])



## 인코딩 설정
1. Tools
2. Global Options
3. Code
4. Saving
5. Default text encoding [Change...]
6. UTF-8
   


## R script 생성하기
1. File 
2. New File
3. R script     (Ctrl + Shift + N)

## 코드 실행
- 한 줄 실행        : Ctrl + Enter
- 여러 줄 실행      : 드래그 → Ctrl + Enter
- 스크립트 실행     : Ctrl + Alt + R
